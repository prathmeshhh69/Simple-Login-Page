# Simple React Login Page

## Overview

This project is a basic **Login Page UI** built using **React** and **Tailwind CSS**.
It demonstrates form layout design, input handling, and basic component structure for authentication pages.

## Features

* Clean and responsive login UI
* Username / Email and Password input fields
* Styled using Tailwind CSS utility classes
* Reusable component structure for easy integration into larger projects
* Ready to integrate with backend authentication APIs

## Tech Stack

* React (Functional Components)
* Tailwind CSS
* JavaScript (ES6)

## Installation

1. Clone the repository

   ```bash
   git clone <your-repo-link>
   ```

2. Navigate to the project folder

   ```bash
   cd <project-name>
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the development server

   ```bash
   npm run dev
   ```
